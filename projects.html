看看我的 项目经验

以下项目来自我的真实简历，覆盖仓储供应链、生产制造、电商、财务管理和社区 APP。

2025.06 - 至今 01

WMS仓储供应链系统

面向智能制造企业的仓储管理系统，覆盖出入库流程、库位优化、动态盘点等功能。

测试工具 XMind、MySQL、Navicat、JMeter、Fiddler
  • 参加项目需求分析，结合仓储业务流程提出需求疑问。
  • 根据接口文档使用 XMind 编写测试点、接口测试用例，并参与用例评审。
  • 使用 JMeter 执行接口测试用例，验证接口参数、业务逻辑和返回结果。
  • 负责 WMS 系统功能测试、接口测试及 JMeter 接口自动化测试。
  • 参与需求分析，搭建测试环境，协助开发进行冒烟测试。
  • 覆盖入库、库存、出库、波次管理等核心模块，从 0 到 1 完成测试。
  • 利用 Fiddler 工具对 PDA 端请求和响应进行抓包分析。
  • 使用 JMeter 对发货订单、生成拣货单等接口进行并发性能测试。
  • 执行 WMS 与 ERP、TMS、MES 外部系统 API 测试，并验证数据一致性。
  • 执行主要模块回归测试，验证缺陷是否修复，同时关注是否产生新问题。
  • 汇总测试过程、缺陷情况和质量风险，完成测试报告输出。
